# Python DSA

A collection of data structures and algorithms implemented from scratch in Python as part of my Data Structures & Algorithms (DSA) learning journey.

The goal of this repository is to understand how common data structures and algorithms work internally by implementing them without relying on Python's built-in data structures whenever possible.
